Prototipo frontend - Adopta un Amigo
Archivos incluidos:
- index.html (Home)
- adopta.html (Lista de animales y solicitud de adopción)
- contacto.html (Formulario de contacto / voluntariado)
- styles.css (Estilos)
- scripts.js (Comportamiento JS)
- img/ (carpeta con imágenes de ejemplo - se incluyen imágenes de ejemplo en blanco si no hay recursos)

Cómo usar:
1. Descomprime el archivo y abre index.html en tu navegador.
2. El prototipo es estático; los formularios simulan envío con alertas (útil para demostración y evaluación).
3. Recomendado: abrir en Chrome/Firefox para mejor visualización.

Accesibilidad y pruebas aplicadas:
- Contraste de color razonable para textos y botones.
- Elementos de formulario con labels y focus visible.
- Modal con roles y atributos aria-modal.
- Navegación por teclado (tab) y cierre con ESC.
